``1-my_list`` module test
=============================

Using ``MyList`` class
-------------------------------

This text file contains test cases for the class ``MyList``.
First import class to the function print_sorted:
    
    >>> MyList = __import__('1-my_list').MyList

    Now use it:

    Test_1 print my_list
    ---------------------------

     >>> my_list = MyList()
     >>> my_list.append(1)
     >>> my_list.append(4)
     >>> my_list.append(2)
     >>> my_list.append(3)
     >>> my_list.append(5)
     >>> my_list.print_sorted()
     [1, 2, 3, 4, 5]


    Test_2 print list empty
    -------------------------
    
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

    Test_3 print list with negative
    -------------------------------
    
    >>> my_list = MyList()
    >>> my_list.append(-3)
    >>> my_list.append(-1)
    >>> my_list.append(-5)
    >>> my_list.append(-2)
    >>> my_list.append(-4)
    >>> print(my_list)
    [-3, -1, -5, -2, -4]
    >>> my_list.print_sorted()
    [-5, -4, -3, -2, -1]
    >>> print(my_list)
    [-3, -1, -5, -2, -4]

    test 4 list with negative and positive numbers
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(3)
    >>> my_list.append(-1)
    >>> my_list.append(-2)
    >>> my_list.append(4)
    >>> print(my_list)
    [5, 3, -1, -2, 4]
